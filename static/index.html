About

Latrunculi (or ludus latrunculorum) is a two-player strategy board game that dates back over two thousand years. Much of the game's history, along with the specifics of its rules, have been lost over time. Like the original game, this web version of the game requires an opponent. After creating a new game you can send the game link to a friend, or just open it in a second window to try it out on your own.

Learn more about the history of Latrunculi here.

Rules

The original rules of latrunculi have been lost in history, so this version chooses to follow Kowalski's reconstruction of the game rules. These rules are simple:

  • Each piece may move any unobstructed distance along a rank (column) or file (row).
  • A man (the circular piece) is captured if the enemy places a piece adjacent to it on each side in an orthogonal line.
  • If a man is moved voluntarily between two enemy pieces, it is not captured.
  • A man in a corner is captured if the opponent places his men on the two squares adjacent to the corner.
  • The dux (the triangular piece) cannot be captured. It is immobilized if blocked on all four sides. A player who immobilizes the enemy's dux wins the game, even if some of the obstruction is by the dux's own men.
